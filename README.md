# C4C


Node.js のインストール
> https://nodejs.org/en/download/


npmの初期化
> npm init


ejsのインストール
> npm install -g ejs


expressのインストール
> npm install -g express


express-generatorのインストール
> npm install -g express-generator


watsonのインストール
> npm install -g ibm-watson
> npm link ibm-watson


ローカルでWebサーバを起動
> express myapp
> cd myapp
> npm install
> npm start

検索サイトを開く
> http://localhost:3000/form/
